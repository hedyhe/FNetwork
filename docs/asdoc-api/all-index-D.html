DataBoxChangeEvent — class, package com.myflexhero.network.event
数据操作事件类,将统一派发type="dataBoxChange"类型的事件。 请通过监听DataBox(子类有ElementBox、LinkBox、AlarmBox等)类来截获该事件的派发，并通过DataBoxChangeEvent的kind属性判别操作类型。 已有的kind属性有如下所示值： DataBoxChangeEvent.ADD 新增指定的数据元素。调用DataBox.add()方法和DataBox.setDatas()方法(每一个数据元素将派发一次)将派发该事件，并设置DataBoxChangeEvent的data属性为新增的数据元素。 DataBoxChangeEvent.REMOVE 删除指定的数据元素。调用DataBox.removeData()方法将派发该事件，并设置DataBoxChangeEvent的data属性为删除的数据元素。 DataBoxChangeEvent.CLEAR 清空当前所属的数据容器所有的数据元素。调用DataBox.removeAll()方法将派发该事件，并设置DataBoxChangeEvent的datas属性为已删除的数据集合。 DataBoxChangeEvent.CHECK 内部使用，目前无需自行处理。可参阅ElementBox.check属性。
