Grid — class, package com.myflexhero.network

Group — class, package com.myflexhero.network
Group组件将一些Node节点(及其子类)看作是一个内部组， 并绘画出一个椭圆(通过指定Styles.GROUP_SHAPE样式可使用常量中以Consts.SHAPE_开头的形状)的边界， 该边界的宽和高值由内部子类的最小、最大x、y值决定。 如果内部子类的最小和最大的x、y值发生了改变，GroupUI将重绘其边界。 数据元素的继承结构如下所示: Data(数据基类) |Element(元素基类) |Node(节点类) |Follower(具有跟随功能的类) |SubNetwork(具有显示多层次子类关系的类) |AbstractPipe(具有内部显示不同形状的抽象类) |RoundPipe(圆管,内部可显示父子关系的圆孔) |SquarePipe(矩形管道,内部可显示不同大小的矩形) |Group(内部作为一个整体，具有统一的边界外观) |Grid(网格，可设置任意行、列) |KPICard(可表示KPI的网格) |Link(链接类，表示节点之类的关系)
