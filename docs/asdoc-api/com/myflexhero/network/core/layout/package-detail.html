Classes
 ClassDescription
 AutoLayout 自动布局控制器.
 BasicLayout 所有布局的基类，继承自EventDispatcher，使其可以派发事件
 CustomLayout 自定义布局的基类,提供了部分通用的方法。
 RoundLayout 
 SpringLayout 弹力环形布局，调整因子主要有3个： repulsionFactor 弹力因子,调整各个节点间的间隔,默认值为0.95,可自定义更改. rigidity 晃动因子，设置节点调整期间每次晃动的大小，值越大调整效果越快,默认值为1,暂不建议更改. motionLimit 调整因子，值越小调整时间越长，默认值为0.6,暂不建议更改. 另外，通过设置布尔类型属性autoFit，我们可以为一个屏幕内可以完整显示的元素提供自动调整位置而不出现滚动条的布局优化。
 TreeLayout 树形布局.拥有以下四种布局方向。 Consts.LAYOUT_TOPBOTTOM Consts.LAYOUT_BOTTOMTOP Consts.LAYOUT_LEFTRIGHT Consts.LAYOUT_RIGHTLEFT 可以使用的参数有:orientation、depthSpace、 leftGap、 topGap、siblingSpace。 仍需修改以支持多个顶级节点群。
