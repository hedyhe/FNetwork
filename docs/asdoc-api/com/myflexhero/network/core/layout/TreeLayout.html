Packagecom.myflexhero.network.core.layout
Classpublic class TreeLayout
InheritanceTreeLayout CustomLayout BasicLayout flash.events.EventDispatcher

树形布局.拥有以下四种布局方向。
  • Consts.LAYOUT_TOPBOTTOM
  • Consts.LAYOUT_BOTTOMTOP
  • Consts.LAYOUT_LEFTRIGHT
  • Consts.LAYOUT_RIGHTLEFT
可以使用的参数有:orientation、depthSpace、 leftGap、 topGap、siblingSpace。 仍需修改以支持多个顶级节点群。
