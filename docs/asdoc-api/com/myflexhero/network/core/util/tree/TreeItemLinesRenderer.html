Classpublic class TreeItemLinesRenderer
InheritanceTreeItemLinesRenderer mx.controls.treeClasses.TreeItemRenderer

Extends TreeItemRenderer to draw the dotted lines of the tree. It supports 3 line styles - dotted (default), solid, or none. And the tree line color, alpha, and thickness values are configurable styles too. 
	  <ui:TreeItemLinesRenderer
	   Styles
	   lineAlpha="1"
 	   lineColor="#808080"
 	   lineThickness="1"
	   lineStyle="dotted"
 	 >
 	      ...
 	      child tags
 	      ...
 	  </ui:TreeItemLinesRenderer>
