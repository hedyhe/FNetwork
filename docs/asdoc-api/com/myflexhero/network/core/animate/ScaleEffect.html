Packagecom.myflexhero.network.core.animate
Classpublic class ScaleEffect
InheritanceScaleEffect Object

放大缩小界面的工具类，供Network直接调用。可以自定义更改SCALE_BY值(默认为0.2,即每次放大缩小原始大小的1/5大小)和minScaleBy(所允许的最低缩小倍数,默认为0.01)。
