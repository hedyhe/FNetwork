是否将所有元素添加到一个名为child的UIComponent组件中，然后将该组件添加到当前容器中。 如果需要设置当前容器的布局器为tileLayout，需要设置useDefaultChild为false，使所有子元素跳过名为child的UIComponent组件，而直接添加到当前容器中。
