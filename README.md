# FNetwork

FNetwork is Flex/Flash based professional telecommunication UI component in this field

开发使用
-----------------------------------  
1、FNetwork是可支持大规模节点拓扑关系运算的图形组件，主要支持拓扑关系展示、伪2D和逼真2D房间等展示
2、代码开发于2013年中旬，当时运行正常，但目前的Flex版本较新，所以代码可能存在需升级的小BUG，待完善
3、组件的开发使用文档请参见/docs/asdoc-api下的[index.html](http://htmlpreview.github.io/?https://github.com/ihedy/FNetwork/blob/master/docs/asdoc-api/index.html)<br />
4、更多示例参见FNetworkTest工程，类似如下：<br />
